# Tetris Game

This is a simple implementation of the classic Tetris game built with HTML, CSS, and JavaScript. It features a modern design with a black background, grey grid lines, and colorful Tetrominoes with white borders. You can play the game directly in your browser.

## Features
- **Classic Tetris Gameplay**: The game is based on the original Tetris mechanics where you control falling Tetrominoes and try to fill up complete lines to score points.
- **Responsive Controls**: Use arrow keys to move and drop the Tetrominoes.
- **Start Screen**: A pop-up screen to start the game.
- **Game Over Screen**: Shows the final score and provides an option to restart the game.
- **Score Tracking**: Your score increases as you complete full lines.
- **No Rotation**: The Tetrominoes do not rotate, and you can only move them left, right, or down.

## How to Play
- **Move Left**: Press the Left Arrow Key (`←`).
- **Move Right**: Press the Right Arrow Key (`→`).
- **Move Down**: Press the Down Arrow Key (`↓`).
- **Start the Game**: Click the "Start" button on the pop-up screen.
- **Restart the Game**: When the game ends, click the "Restart" button in the Game Over pop-up.
